# WIDS Movie Recommendation System

This repository contains Jupyter Notebook assignments focused on **data preprocessing and feature engineering** for a movie recommendation system.

## Files
|── Assignment 1.ipynb
├── Assignment 2.ipynb
├── Assignment 3.ipynb
└── README.md

---
## Tasks Completed
-Imported the dataset using pandas 
-Counted number of movies in each genre and represented it in form of a bar graph
-Merged 3 datasets to form a master dataset for further operations
-Extracted keywords from each movie with help of stemming
-Sorted each movie based on popularity

## Assignment Summary

**Assignment 1:** Dataset exploration and understanding movie metadata  
**Assignment 2:** Data cleaning and feature extraction (cast, director, keywords)  
**Assignment 3:** Feature engineering and keyword analysis for recommendations
